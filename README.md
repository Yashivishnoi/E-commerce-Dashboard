
# E-Commerce Dashboard

A modern admin dashboard for managing products in an e-commerce setup. It features product CRUD (Add/Edit/Delete/Search), real-time analytics with interactive charts, QR code generation for products, voice search, dark mode toggle, and CSV export functionality. The dashboard provides a clean, user-friendly interface to efficiently manage and track inventory.

## Features
- Add, edit, delete, and search products
- Category-wise analytics with Chart.js
- QR code generation for each product
- Voice search to quickly find products
- Dark mode toggle for comfortable viewing
- Export product list to CSV

## Tech Stack
- HTML, CSS, JavaScript
- Chart.js for interactive analytics
- QRCode.js for product QR codes
- Local Storage for data persistence

## How to Run
1. Clone the repository:  
   `git clone https://github.com/Yashivishnoi/E-commerce-Dashboard.git`
2. Open `index.html` in your browser.  



**Note:** This dashboard is fully front-end based and stores data locally in the browser using Local Storage. For production use, a backend with a database is recommended.
